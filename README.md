# Euro-Art Responsive Website

A modern, production-ready responsive website for the "Euro-Art" advertising agency, built using the required tech stack and adhering strictly to provided guidelines.

## Tech Stack
- **Vite**
- **React** (TypeScript)
- **Tailwind CSS v4**
- **Bun** (for package management)

## Features
- **Responsive Design**: Works perfectly across mobile, tablet, and desktop.
- **Modern Aesthetic**: Clean UI, subtle shadows, rounded corners, and CSS-first micro-interactions.
- **SEO Optimized**: Includes meta tags, Open Graph tags, and a valid title.
- **Performance**: Lazy loading for images, highly optimized Vite build, lightweight components.
- **Accessibility**: Semantic HTML and global respect for `prefers-reduced-motion` settings.

## Commands
This project uses Bun. The following commands are supported:

- `bun install`: Install dependencies.
- `bun dev`: Start the development server.
- `bun run build`: Create a production bundle.
- `bun run preview`: Preview the production build locally.
